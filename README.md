# Quiz EPICODE | Benchmark Exam (Build Week)

Web app single-page che simula l’esperienza del “Benchmark Exam” EPICODE: Welcome, quiz a tempo con domande a scelta multipla e schermata risultati.

## Funzionalità principali

- Single-page flow: Welcome → Quiz → Results
- Domande a scelta multipla (una sola risposta)
- Timer per domanda con ring circolare
- Punteggio finale e schermata risultati
- UI responsive (desktop e mobile)
